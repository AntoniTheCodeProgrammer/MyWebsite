Dino but Better

Gra arkadowa w PyGame

Projekt gry zręcznościowej inspirowanej klasycznym Dino z Google Chrome, ale z ulepszoną mechaniką, grafiką i nowymi przeszkodami. Gra została napisana w języku Python z użyciem biblioteki PyGame.

Zobacz na GitHub

Główne cechy:

  • Nowoczesne mechaniki

    Ulepszony system skoków, kucania i wykrywania kolizji dla bardziej płynnej rozgrywki.

  • Dynamiczna trudność

    Prędkość gry rośnie wraz z czasem, wprowadzając coraz trudniejsze wzorce przeszkód.

  • System zapisu punktów

    Twój rekord jest zapisywany lokalnie, abyś mógł go pobić przy następnej sesji.
